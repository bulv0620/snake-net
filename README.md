# 🐍联机贪吃蛇
## JavaScript技术栈

- 后端: express + socket.io
- 前端: Vite + Vue3 + socket.io-client

## 示例地址

`http://snake.imbulv.cloud`

## 前端

```
cd snake-net-front
npm install # 安装依赖
npm run dev # 调试运行
npm run build # 打包
```

## 后端

```
cd snake-net-server
npm install  # 安装依赖
npm run dev # 调试运行
```
